About This Project

The Bassa-XAU Trader is a demonstration of my ability to create sophisticated financial trading interfaces with real-time data visualization. This application showcases:

  • Real-time price tracking for XAU/USD (Gold vs USD)
  • Interactive trading simulation with demo account
  • Professional chart integration using TradingView
  • Responsive design that works on all devices
  • Dark/light mode toggle for user preference
  • User authentication system
  • Transaction history tracking
  • Offline functionality
